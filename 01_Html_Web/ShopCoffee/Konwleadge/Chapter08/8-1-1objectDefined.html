3. 基本数据类型

基本数据类型：Number、String、boolean、undefined、NULL(空对象 object)

引用数据类型：Array Function Object

基本数据类型 只传递值

var x=50;
var y=x;
var y=30;
console.log(x);
console.log(y);

引用数据类型 只传递存储变量的地址

var arr=[1,2,3];
var arrs=arr;
arr[1]=30;
console.log(arr);
console.log(arrs);
