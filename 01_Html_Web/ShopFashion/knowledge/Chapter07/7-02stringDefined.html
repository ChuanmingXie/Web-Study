7-01 String对象的声明定义

7-01-1 String对象的创建

语法：

字面量方式
var name = "漫步时代广场";
var address = "中国 青岛 高新区";
new 方式
var movieName = new String('我和我的父辈-如愿');
var director = String("陈凯歌-吴京");

7-01-2 string与String的区别

区分：

var str1="abc";
var str2="abc";
var objectStr1=new String('abc');
var objectStr2=new String('abc');
str1=str2 返回 true;
objectStr1=objectStr2 返回 false;

示例：

var encourage='美丽的';
encourage.describe='程序员指导师';
var team=new String('好斗的');
team.describe='程序员团队';
因为 encourage 不是对象，则显示undefined
team.describe 则正常显示
