    在HTML4中浏览器的主要功能是负责HTML内容的呈现，当客户端有少量的数据存储时，需要通过cookie技术实现，Cookie有一下几个缺点：
        A. Cookie的小被限制在4KB以内
        B. Cookie会随着Http请求一起发送到服务器，重复多次的发送会导致带宽的浪费。
        C. Cookie信息在网络传输的过程中并未进行加密，存在一定的安全隐患
        D. Cookie操作相对复杂
